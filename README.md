# LT10 Gest-ON-AIR

Prototipo de interfaz interna construido con **Angular 20**, **Tailwind CSS** y **PrimeNG** para replicar el modelo de maqueta del sistema Gest-ON-AIR de LT10.

## Scripts

```bash
npm install   # instala dependencias
npm start     # levanta la app en modo desarrollo
npm run build # compila para producción
```

## Características
- Layout oscuro inspirado en el manual de identidad visual de LT10.
- Componentes autónomos para Dashboard, Clientes, Órdenes y Programación diaria impulsados por **Signals** y el nuevo control flow nativo.
- Formularios reactivos con controles PrimeNG.
- Configuración completa de Tailwind para personalizar la paleta de color institucional.
